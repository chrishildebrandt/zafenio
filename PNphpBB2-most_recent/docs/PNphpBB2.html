Part 3 : The Attachment Mod

PNphpBB2 incorporates (and will keep up to date with) the Attachment Mod from DSCS Opentools http://www.opentools.de/. Documentation for using this mod can be found here (Attachment Mod User Guide). Or from DSCS Opentools site. Please omit the installation sections 3.1 & 3.2 of the guide. The mod is already installed into PNphpBB2.

Direct all support question to PNphpBB2 http://www.pnphpbb.com as DSCS Opentools will not support installations that are not using the core phpBB2 forum.

Note: The complete Attachment mod user guild is included but most of 3 and 9 do not apply to the PNphpBB2 module.
